iOS 获取 iPod 音乐库歌曲文件信息并拷贝文件到个人的 App 中

最近有需要从 iOS 系统的音乐库里面拷贝歌曲到 APP 的 Documents 目录下进行共享使用，于是总结一下使用的方法。其中使用了 MPMediaQuery 来获得音乐文件的信息，比如歌手、专辑名、专辑封面、时长、 ASSetURL 地址等，然后通过 ASSetURL 地址使用 AVAssetExportSession 导出其数据。

一. 访问 MediaLibrary

Apple官方文档指出，访问iPod Library的方法有两种，分别是 MediaPicker 和 MediaQuery。

MediaPicker

MediaPicker是一个高度封装的iPod Library访问方式，通过使用 MPMediaPickerController类来访问iPod Library。这是一个UI控件，用户可以根据需要选择其中的音乐。这个类使用时非常方便，只需要生成一个 MPMediaPickerController 的实例，设置一下属性和 delegate 后 present 出来，接下来只要等待回调即可，在回调时需要手动 dismiss picker。

 1 - (void)convertMediaPickerController{
 2 
 3     //MPMediaPicker
 4     MPMediaPickerController *picker = [[MPMediaPickerController alloc] initWithMediaTypes:MPMediaTypeAnyAudio];
 5     picker.prompt = @"请选择需要播放的歌曲";
 6     picker.showsCloudItems = NO;
 7     picker.allowsPickingMultipleItems = YES;
 8     picker.delegate = self;
 9     [self presentViewController:picker animated:YES completion:nil];
10 }

得到的效果图如下：

通过 MediaPicker 最终可以得到 MPMediaItemCollection，其中存放着所有在Picker中选中的歌曲，每一个歌曲使用一个 MPMediaItem 对象表示。对于MediaPicker的使用也可以参考Apple官方文档。

MediaQuery

如果觉得 MeidaPicker 的功能或者 UI 不能满足你的要求那么可以使用 MediaQuery。MediaQuery 可以直接访问 iPod Library 的数据库，并根据需求获取数据。详情可以参见Apple官方文档

MediaQuery 功能十分强大，它可以根据一个或多个条件查询满足需要的 MediaItem。

你可以使用 MPMediaQuery 的类方法来生成一些已经预置了条件的 Query

也可以自己生成 MPMediaPredicate 设置条件，并把它加到 Query 中，最后通过 items 和 collections 访问查询到的结果，例如：

 1 MPMediaPropertyPredicate *artistNamePredicate =
 2 [MPMediaPropertyPredicate predicateWithValue:@"Happy the Clown"
 3                                  forProperty:MPMediaItemPropertyArtist
 4                               comparisonType:MPMediaPredicateComparisonEqualTo];
 5 
 6 MPMediaQuery *quert = [[MPMediaQuery alloc] init];
 7 [quert addFilterPredicate: artistNamePredicate];
 8 quert.groupingType = MPMediaGroupingArtist;
 9 
10 NSArray *itemsFromArtistQuery = [quert items];
11 NSArray *collectionsFromArtistQuery = [quert collections];

mediaQuery

 2 
 3 /**
 4  *  读取音乐文件
 5  *
 6  *  @param completion 完成后返回一个包含所有音乐信息的 array，
 7  */
 8 - (void)readMusicItemsFromeAlbumCompletion:(void (^)(NSMutableArray *musicArray, NSError *error))completion{
 9     __block NSMutableArray *musicItemsArray = [NSMutableArray array];
10     __block NSError *err = nil;
11     
12     MPMediaPropertyPredicate *predicate = [MPMediaPropertyPredicate predicateWithValue:@(MPMediaTypeAnyAudio) forProperty:MPMediaItemPropertyMediaType];
13     MPMediaQuery *mediaQuery = [[MPMediaQuery alloc] initWithFilterPredicates:nil];
14     [mediaQuery addFilterPredicate:predicate];
15 
16     NSArray *items = [mediaQuery items];
17     
18     for (int i = 0; i < [items count]; i++) {
19         
20         // 保存所有音乐信息
21         VNBMusicItem *musicItem = [[VNBMusicItem alloc] initWithMPMediaItem:items[i]];
22 
23         // 不显示Apple Music里面下载的音乐
24         /**
25          *  由于iPhone 自带的音乐软件Music的推出.从iPod取出来的音乐MPMediaItemPropertyAssetURL属性可能为空.
26          *  这是因为iPhone自带软件Music对音乐版权的保护,对于所有进行过 DRM Protection(数字版权加密保护)的音乐
27          *  都不能被第三方APP获取并播放.即使这些音乐已经下载到本地.但是还是可以播放本地未进行过数字版权加密的音乐.
28          *  也就是自己手动导入的音乐.
29          */
30         if (musicItem.assetUrl) {
31             [musicItemsArray addObject:musicItem];
32         }
33     }
34     
35     if (completion) {
36         completion(musicItemsArray, err);
37     }
38 }
39 @end

44     self = [super init];
45     if (self) {
46         
47         self.audioItem = item;
48 
49         //URL 地址
50         self.assetUrl = [item valueForProperty:MPMediaItemPropertyAssetURL];
51         //歌手
52         self.artistName = [item valueForProperty:MPMediaItemPropertyArtist];

61         self.albumName = [item valueForProperty: MPMediaItemPropertyAlbumTitle];

65         if (artwork!=nil) {

67         }
68         //播放时长

70         //时长
71         self.IntDuration = [self getIntDurationOfAudioWithItem:item];
72         
73     }
74     return self;

MediaItem

通过 MediaPicker 和 MediaQuery 最终都会得到 MPMediaItem，这个item中包含了许多信息。这些信息都可以通过 MPMediaEntity 的方法访问，其中参数非常多，具体可以参照MPMediaItem.h。

导出数据

前面代码使用 MPMediaItem 的 MPMediaItemPropertyAssetURL 属性可以得到一个表示当前 MediaItem 的NSURL，有了这个NSURL 我们便可以使用 AVFoundation 中的类进行播放，也可以使用 AVFoundation 中两个有趣的类：AVAssetReader和AVAssetExportSession，它们可以把 iPod Library 中的指定歌曲以指定的音频格式导出到内存中或者硬盘中，这个指定的格式包括 PCM。这是一个非常炫酷的特性，有了 PCM 数据后我们就可以实现很多很多功能了。

这部分如果要展开的话还会有相当多的内容，国外的先辈们早在2010年就已经发掘了这两个类的用法，详细参见这里和这里。这两篇讲的比较详细并且附有Sample（其中还涉及了一些Extended Audio File Services的内容），如果里面Sample无法下载可以从点击MediaLibraryExportThrowaway1.zip 和 VTM_AViPodReader.zip 下载。

示例代码

下面的代码就是利用了 AVAssetExportSession 将 iPod 库中的歌曲数据导出为 m4a 文件：

  1 #define Cache_Directory [[NSSearchPathForDirectoriesInDomains(NSDocumentDirectory, NSUserDomainMask, YES) objectAtIndex:0] stringByAppendingPathComponent:@"Web"]
  2 
  3 @interface VNBMusicManager ()
  4 
  5 @property (nonatomic, strong) AVAssetExportSession* exportSessionMusic;
  6 
  7 @end
  8 
  9 @implementation VNBMusicManager
 10 
 11 /**
 12  *  准备共享音频：先把音频文件从 ipod 库中拷贝出来，再上传给电视播放
 13  *
 14  *  @param item       需要共享的音乐对象信息
 15  *  @param error      错误信息
 16  *  @param completion 完成后开始共享
 17  *
 18  *  @return 是否读取完成
 19  */
 20 - (BOOL)prepareToShareAlbumItem:(VNBMusicItem *)item error:(NSError **)error completion:(void (^)(NSError *err))completion{
 21     
 22     __block NSError *err = nil;
 23     
 24     NSFileManager *fm = [NSFileManager defaultManager];
 25     //若文件夹不存在，创建文件夹
 26     if (![fm fileExistsAtPath:Cache_Directory]) {
 27         [fm createDirectoryAtPath:Cache_Directory withIntermediateDirectories:YES attributes:nil error:error];
 28         if (*error) {
 29             NSLog(@"创建文件夹失败：%@", *error);
 30             return NO;
 31         }
 32     }
 33     
 34     NSString *localPath = [Cache_Directory stringByAppendingPathComponent:item.fileName];
 35 
 36     [self clearCacheWithOutMusicItem:item];
 37     
 38     //音乐 m4a
 39     NSString *outPathUrl = [[Cache_Directory stringByAppendingPathComponent:item.trackName] stringByAppendingPathExtension:@"m4a"];
 40     
 41     AVURLAsset *urlAsset = [AVURLAsset assetWithURL:item.assetUrl];
 42     
 43     self.exportSessionMusic = [AVAssetExportSession exportSessionWithAsset:urlAsset presetName:AVAssetExportPresetAppleM4A];
 44     self.exportSessionMusic.outputFileType = AVFileTypeAppleM4A;
 45     self.exportSessionMusic.shouldOptimizeForNetworkUse = YES;
 46     self.exportSessionMusic.outputURL = [NSURL fileURLWithPath:outPathUrl];
 47     
 48     [self.exportSessionMusic exportAsynchronouslyWithCompletionHandler:^{
 49         
 50         switch (self.exportSessionMusic.status) {
 51             case AVAssetExportSessionStatusUnknown:{//0
 52                 NSLog(@"exportSession.status AVAssetExportSessionStatusUnknown");
 53                 break;
 54             }
 55             case AVAssetExportSessionStatusWaiting:{//1
 56                 NSLog(@"exportSession.status AVAssetExportSessionStatusWaiting");
 57                 break;
 58             }
 59             case AVAssetExportSessionStatusExporting:{//2
 60                 NSLog(@"exportSession.status AVAssetExportSessionStatusExporting");
 61                 break;
 62             }
 63             case AVAssetExportSessionStatusCompleted:{//3
 64                 NSLog(@"exportSession.status AVAssetExportSessionStatusCompleted");
 65                 
 66                 //do something
 67                 NSError *error;
 68                 [fm moveItemAtPath:outPathUrl toPath:localPath error:&error];
 69                 //只能导出为 m4a 文件，导出后再更名为 MP3
 70                 //outPathUrl: ~/Documents/Web/musicName.m4a
 71                 //localPath: ~/Documents/Web/musicName.mp3
 72 
 73                 dispatch_sync(dispatch_get_main_queue(), ^{
 74                     if (completion) {
 75                         completion(err);
 76                     }
 77                 });
 78                 
 79                 break;
 80             }
 81             case AVAssetExportSessionStatusFailed:{//4
 82                 
 83                 NSLog(@"exportSession.status2 AVAssetExportSessionStatusFailed");
 84                 if (completion) {
 85                     completion(self.exportSessionMusic.error);
 86                 }
 87                 NSLog(@"error:%@",self.exportSessionMusic.error);
 88                 break;
 89             }
 90             case AVAssetExportSessionStatusCancelled:{//5
 91                 NSLog(@"exportSession.status2 AVAssetExportSessionStatusCancelled");
 92                 break;
 93             }
 94             default:
 95                 break;
 96         }
 97     }];
 98     
 99     return YES;
100 }
101 
102 /**
103  *  清除缓存
104  *
105  *  @param item 准备上传的音乐对象
106  */
107 -(void)clearCacheWithOutMusicItem:(VNBMusicItem *)item{
108     NSFileManager *fm = [NSFileManager defaultManager];
109     NSError *errors = nil;
110     NSArray *fileList = [[NSArray alloc] init];
111     fileList = [fm contentsOfDirectoryAtPath:Cache_Directory error:&errors];
112     for (NSString *object in fileList) {
113         NSString * filePath=[Cache_Directory stringByAppendingPathComponent:object];
114         BOOL Ret = [fm fileExistsAtPath:filePath];
115         if (Ret) {
116             NSError *err;
117             [fm removeItemAtPath:filePath error:&err];
118         }
119     }
120 }
121 @end
